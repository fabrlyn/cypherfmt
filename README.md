## Format
- atom
  - literal
    - [x] null
    - [x] bool
    - number
      - [x] integer
      - [x] double
    - [x] string
    - [x] list
    - [x] map
  - [x] parameter
  - [ ] case
  - [ ] count
  - [ ] filter
  - [ ] pattern comprehension
  - [ ] all
  - [ ] any
  - [ ] collection(all, any, none, single) filter
  - [ ] node
  - [ ] expression
  - [ ] function invocation
  - [ ] variable
